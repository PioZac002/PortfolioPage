# Portfolio Page – Piotr Zaćmiński

Personal portfolio website built with React and Vite. Showcases my projects, tech stack, work experience, and contact information.

## Live Demo

Try it out at: https://piozac002.github.io/PortfolioPage/

## Features

- Responsive design (mobile, tablet, desktop)
- Dark / light theme toggle
- Multilingual support (Polish / English)
- Scroll animations
- Project gallery with photo carousel
- CV section (work experience, education, certifications)

## Tech Stack

- React 19
- Vite
- Custom CSS (no CSS framework)

## Getting Started

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
